## 利用規約

### 第1条（適用）
本規約は、アプリ「unie」（以下「本アプリ」）が提供するサービス（以下「本サービス」）の利用条件を定めるものです。ユーザーと本アプリ提供者との間のあらゆる関係に適用されます。

### 第2条（AIアドバイスの性質）
1. 本アプリが提供するAIによる助言・添削等の情報は、技術向上を支援するための参考情報であり、その正確性、完全性、または特定の成果を保証するものではありません。
2. AIのアドバイスに基づいて行われた制作活動の結果について、本アプリ提供者は一切の責任を負いません。

### 第3条（知的財産権）
1. ユーザーが本アプリにアップロードしたイラスト等のデータに関する著作権は、引き続きユーザー本人に帰属します。
2. 本アプリは、ユーザーの明示的な同意がない限り、アップロードされた作品を外部に公開したり、広告等に無断で使用したりすることはありません。

### 第4条（禁止事項）
ユーザーは、他者の著作権を侵害する画像を入力することや、本サービスのシステムに過度な負荷をかける行為を行ってはなりません。

## プライバシーポリシー

### 1. データの収集と利用目的
本アプリは、AIアドバイスの提供およびサービス改善のために、ユーザーがアップロードした画像データおよび利用ログを取得します。

### 2. AI学習に関する特約（安心宣言）
本アプリは、ユーザーがアップロードした画像データを、**当社のAIモデルの再学習や追加学習のために利用することはありません。** データは、アドバイス生成の目的のみに一時的に処理されます。

### 3. 外部APIの利用
本アプリは、AI解析のためにOpenAI等の外部APIを使用する場合があります。送信されるデータは、各APIプロバイダーの「学習に利用しない」規約に基づき、厳重に保護されます。

### 4. データの削除
ユーザーは、本アプリ内の設定を通じて、サーバー上の自身のデータをいつでも削除することができます。
